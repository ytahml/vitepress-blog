# 01-Python 变量

> Python-100-Days 第三天学习笔记

## 📚 核心知识点

### 1. 程序与数据的本质

- **程序 = 数据 + 指令的有序集合**
- **写程序 = 用数据和指令控制计算机做事**
- 计算机硬件五大部件：运算器、控制器、存储器、输入设备、输出设备
- CPU = 运算器 + 控制器
- 程序执行时，数据和指令需加载到**内存**中

### 2. 二进制基础

- 计算机使用**二进制**（逢二进一）计数
- 物理实现：高电压=1，低电压=0
- 数据在计算机内存中均以**二进制形态**存在

### 3. 变量与类型

**变量**：数据的载体，一块用来保存数据的内存空间，值可被读取和修改

#### Python 常用数据类型

| 类型 | 关键字 | 说明 | 示例 |
|------|--------|------|------|
| 整型 | `int` | 可处理任意大小整数，支持多种进制 | `0b100`(二进制)=4, `0o100`(八进制)=64, `100`(十进制), `0x100`(十六进制)=256 |
| 浮点型 | `float` | 小数，小数点位置可变 | `123.456`, `1.23456e2` |
| 字符串型 | `str` | 单引号或双引号包裹的文本 | `'hello'`, `"hello"` |
| 布尔型 | `bool` | 只有 True/False 两种值 | `True`, `False` |

### 4. 变量命名规则

#### 必须遵守的规则

1. 变量名由**字母、数字、下划线**构成，**数字不能开头**
2. Python 是**大小写敏感**的（`A` 和 `a` 是不同变量）
3. **不要与关键字重名**，避开保留字（如：`is`, `if`, `else`, `for`, `while`, `True`, `False`, `int`, `print`, `input`等）

#### 推荐惯例

1. 使用**小写英文字母**，多个单词用**下划线连接**（如：`user_name`）
2. 受保护的变量用**单个下划线开头**（如：`_protected`）
3. 私有变量用**两个下划线开头**（如：`__private`）
4. **见名知意**最重要！

### 5. 变量类型检查与转换

#### 类型检查

```python
type(变量名)  # 返回变量的类型
```

#### 常用类型转换函数

| 函数 | 功能 | 示例 |
|------|------|------|
| `int()` | 转整数，可指定进制 | `int('123')` → 123, `int('100', base=2)` → 4 |
| `float()` | 转浮点数 | `float('123.45')` → 123.45 |
| `str()` | 转字符串 | `str(100)` → '100' |
| `chr()` | 整数转字符（编码→字符） | `chr(100)` → 'd' |
| `ord()` | 字符转整数（字符→编码） | `ord('d')` → 100 |
| `bool()` | 转布尔值 | `bool('hello')` → True, `bool('')` → False |

#### 类型转换注意事项

- `str` 转 `bool`：非空字符串→`True`，空字符串→`False`
- `bool` 转 `int`：`True`→1，`False`→0
- `float` 转 `int`：直接截断小数部分（`123.45`→123）

## ⚠️ 注意事项

1. **变量命名**：
   - 不要用中文变量名（虽然 Python 支持 Unicode）
   - 避免使用 Python 内置函数名（如 `print`, `input`, `str`）
   - 保持命名简洁且有意义

2. **类型转换**：
   - 不是所有字符串都能转成数字（如 `'hello'` 无法转成 `int`）
   - 浮点数转整数会丢失小数部分
   - 注意进制转换时 `base` 参数的使用

3. **布尔值**：
   - 非空字符串、非零数值转 `bool` 都是 `True`
   - 只有空字符串、0、`None` 等转 `bool` 才是 `False`

## 💡 代码示例

### 基础运算

```python
a = 45
b = 12
print(a + b)  # 57
print(a - b)  # 33
print(a * b)  # 540
print(a / b)  # 3.75
```

### 类型检查

```python
a = 100
b = 123.45
c = 'hello, world'
d = True
print(type(a))  # <class 'int'>
print(type(b))  # <class 'float'>
print(type(c))  # <class 'str'>
print(type(d))  # <class 'bool'>
```

### 类型转换

```python
print(float(100))           # 100.0
print(int(123.45))          # 123
print(int('123', base=16))  # 291 (十六进制)
print(int('100', base=2))   # 4 (二进制)
print(chr(100))             # 'd'
print(ord('d'))             # 100
```
